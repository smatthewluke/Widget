# Setup Obsidian Git Sync

First task: Configure Obsidian to sync with this repo.

## Options

1. **Obsidian Git Plugin** (recommended)
   - Install from Community Plugins
   - Set vault path to this repo's `/vault` folder
   - Enable auto-pull on open
   - Enable auto-push on close

2. **Manual Git**
   - Clone repo locally
   - Point Obsidian at `/vault`
   - Pull/push from terminal as needed
